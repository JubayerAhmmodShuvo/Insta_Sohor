Qusetion 1 : How Does javascript Work?

Answer : Computer is an interesting machine but it has its own language, it doesnot understand our language rather we write code in text editor and magically computer do something and turns it into 0 and 1. Understanding how javascript works makes reading and writing code easier and less frustrating and allows one to focus on the logic of one's application instead of fighting with the grammar of the language. All we know that javascript is a single threaded and interpreted language. If I give a JS file to CPU and ask to execute it in browser it won't understand the language rather with the help of javascript engine it translate the language and tells the computer what to do.There are 8 engine in javascript but the fast one is V8 engine which is written in C++. When we give a javascript file, first it does lexical analysis which break a code into token to identify their meaning and these token will get get formed in tree called AST(Abstract Syntax Tree).Once tree is formed it goes to interpreter which translate and read the files line by line. Then compiler works ahead of time to create a translation of what code we have written and it compiles down to language that can be underwstood by our machine. Google came with V8 engine which combining both interpreter and compailer known JIT compiler to make the engine faster.
